Bite 216. Parse an email header

Write a regular expression to extract 4 pieces of information from an email header:

  • From email
  • To email
  • Subject
  • Date sent (without timezone info)

Use re.match or re.search and capturing parenthesis. Return the captured groupdict of the match object.

... Received: from NzI3MDQ (174.37.77.208-static.reverse.softlayer.com [174.37.77.208])
... by mi22.sendgrid.net (SG) with HTTP id 13f5d69ac61.41fe.2cc1d0b

... Subject: A Test From SendGrid
... Message-ID: <1371661773.974270694268263@mf9.sendgrid.net>

def get_email_details(header: str) -> dict:
    """User re.search or re.match to capture the from, to, subject
       and date fields. Return the groupdict() of matching object, see:
       https://docs.python.org/3.7/library/re.html#re.Match.groupdict
       If not match, return None
    """
    if not re.findall("From: (.*)", header):
        return None
    a = re.findall("From: (.*)", header)[0]
    b = re.findall("To: (.*)", header)[0]
    c = re.findall("Subject: (.*)", header)[0]
    d = re.findall("Date: (\w+,\s\d+\s\w+\s\d+\s\d+:\d+:\d+)", header)[0]
    text = a + "|" + b + "|" + c + "|" + d
    result = re.match(
        r"(?P<from>\w.+)\|(?P<to>\w.+)\|(?P<subject>\w.+)\|(?P<date>\w.+)", text
    )
    return result.groupdict()
